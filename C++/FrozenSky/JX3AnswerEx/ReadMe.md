## 剑网三科举小助手(JX3NCEE)

之前回归的时候发现多玩盒子自动科举的黑科技被封印，只能到网页上查，可是来回切换打字太累，于是自己写了这个小玩意。

比多玩的科举查询好的地方在于不需要打汉字，只打拼音首字母即可，且可以跳过标点符号，答题速度大大提高。

另外，还有一些方便快速答题的黑科技：
**窗口置顶** 可以一边看题，一边查询。
**无缝切换** 使用Alt+TAB切换到小助手将自动高亮搜索框，方便搜索下一题。
